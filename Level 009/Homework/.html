Top 5 Programming Languages

Explore the most popular programming languages, their uses, and history.

1. Python

Python is a high-level programming language, created by Guido van Rossum in 1991. It is widely used in web development, data science, artificial intelligence, and automation.

Python is known for its readable syntax, making it a popular choice for beginners and professionals alike.

Example code:

def greet(name):
    print(f"Hello, {name}!")
greet("Python")

2. JavaScript

JavaScript was created by Brendan Eich in 1995. It is the language of the web, used to build dynamic and interactive web pages.

JavaScript works seamlessly with HTML and CSS to create complete web applications.

Example code:

function greet(name) {
    alert("Hello, " + name + "!");
}
greet("JavaScript");

3. Java

Java is a versatile programming language, developed by James Gosling in 1995.

It is used in mobile app development (Android), enterprise software, and web applications. Java's principle is "Write Once, Run Anywhere."

Example code:

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}

4. C++

C++ is a general-purpose programming language created by Bjarne Stroustrup in 1983. It is widely used in game development, high-performance applications, and embedded systems.

C++ combines the efficiency of low-level programming with the features of object-oriented programming.

Example code:

#include 
using namespace std;

int main() {
    cout << "Hello, C++!" << endl;
    return 0;
}

5. Ruby

Ruby is a dynamic programming language created by Yukihiro Matsumoto in 1995. It is best known for powering the Ruby on Rails framework.

Ruby emphasizes developer productivity and simplicity, with a focus on readable code.

Example code:

puts "Hello, Ruby!"
