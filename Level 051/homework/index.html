Document Object Model (DOM): Document Object Model (DOM) არის ვირტუალური წარმოდგენა ვებ-გვერდის სტრუქტურისა და ელემენტების. იგი საშუალებას აძლევს პროგრამისტებს, JavaScript-ის გამოყენებით, მიიღონ წვდომა და მართონ HTML და XML დოკუმენტების სტრუქტურა, შინაარსი და სტილი. DOM ქმნის ხის სტრუქტურას, სადაც თითოეული ელემენტი წარმოადგენს კვანძს (node).

რა არის console: console არის JavaScript-ის ობიექტი, რომელიც გამოიყენება მონაცემების გამოსატანად დეველოპერის ინსტრუმენტების ფანჯარაში (მაგალითად, ბრაუზერში). იგი განსაკუთრებით სასარგებლოა პროგრამის დებაგინგისთვის, რათა დეველოპერს შეეძლოს ცვლადების მნიშვნელობის, შეცდომების ან ლოგიკური პროცესების მონიტორინგი.

რა არის let keyword: let არის JavaScript-ის საკვანძო სიტყვა, რომელიც გამოიყენება ცვლადების განსასაზღვრავად. იგი წარმოადგენს ბლოკის ფარგლებში (block scope) ცვლადებს, რაც ნიშნავს, რომ let-ით განსაზღვრული ცვლადი ხელმისაწვდომია მხოლოდ იმ კოდის ბლოკში, სადაც ის გამოცხადებულია. ეს უფრო მოქნილია, ვიდრე var, რომელიც ფუნქციის ფარგლებში მუშაობს.
