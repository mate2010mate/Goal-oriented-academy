Ternary operator (? :): It's a shorthand for an if-else statement. It checks a condition and returns one value if the condition is true, and another if the condition is false.

Syntax: condition ? valueIfTrue : valueIfFalse;
Example: let result = (x > 10) ? "Greater" : "Smaller";
